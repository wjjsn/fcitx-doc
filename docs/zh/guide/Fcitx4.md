Fcitx [ˈfaɪtɪks] 是一个支持扩展的输入法框架。当前它支持 Linux 和 Unix 系统，例如 freebsd。它包括了三个内置的输入法引擎，拼音，区位和基于码表的输入法。

Fcitx 尝试提供在所有桌面环境下原生的体验，并保持其内核轻量。你可以很简单的将其按照你的需要进行配置。
