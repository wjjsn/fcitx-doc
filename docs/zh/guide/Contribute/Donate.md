# 捐赠

出于隐私考虑，对于所有捐赠我们只会在此处列出日期和金额(捐赠历史)。 

所有Fcitx开发者都在利用业余时间开发Fcitx。所以捐赠一些钱对于他们来说将会是一个非常大的鼓励。

目前，我们没有集中账户来筹集资金，因此向 Fcitx 捐款的唯一方式是向个人开发者捐款。

所有开发人员请随时在此处添加您的名字。为了保护隐私，所有电子邮件地址不应直接显示。如果您想查找此处未列出的电子邮件帐户，您也可以使用您的 Google 帐户浏览 (http://code.google.com/p/fcitx Fcitx 谷歌代码主页)，然后单击各个页面上的开发者和贡献者，您可以轻松找到他们的电子邮件帐户。

由于 github 上正在进行一些其他开发，您也可以简单地克隆代码，并通过 git 在 (http://github.com/fcitx/ Fcitx Github 组织) 查看电子邮件帐户记录
