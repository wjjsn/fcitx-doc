# History

Fcitx was developed from a project called gWubi created by Yuking. Yuking himself is a Wubi input method user, so initially fcitx only supported the Wubi input method.

Initially, gWubi didn't use SCM for source code control, so early code history can only be found in old tarball archives. ZhouShuqun still maintains a mirror of all fcitx source code tarball archives, which can be downloaded from [here](https://mirrors.redv.com/pub/fcitx/).

Zhoushuqun added the initial fcitx code submission record on Google Code's subversion, which was almost the only tarball source. When Yuking restarted Fcitx development, he also started using Google Code, so [this Google Code repository](https://code.google.com/p/fcitx) became the official site.

Fcitx's source control changed from the initial Subversion, to Mercurial, and finally to Git. Since Google Code is no longer the best Git hosting site, [CSSlayer](https://github.com/csslayer) decided to use [github.com](https://github.com/fcitx/) for future Fcitx development.

## The Story Behind the Name Fcitx

The old full name of Fcitx was "Free Chinese Input Tool of X" as an acronym. This concept was not suitable for Fcitx's future development and might cause misunderstandings because Chinese is not the only target of Fcitx. So [CSSlayer](https://github.com/csslayer) started a [rename discussion](https://groups.google.com/forum/) on fcitx-dev Google groups, keeping Fcitx as the name but using a different full name.

At the same time, there were complaints that Fcitx couldn't be pronounced, so [ˈfaɪtɪks] was decided as the official pronunciation. However, "Little Penguin Input Method" is still the official Chinese name. Although the Chinese name has no relation to the full name in meaning, it's simply because of the penguin logo.

So Fcitx will not have a single official full name, but you can pick any of the following names to reflect various aspects of Fcitx:

- Abbreviation for Flexible Context-aware Input Tool with eXtension
- Abbreviation for FanCy Input Tool for *niX
- Abbreviation for Forthcoming CJK Input Toy Xtreme
- Abbreviation for Free Character Input Toy of *niX
- Abbreviation for Fcitx Can Input Text in *niX

## Controversy
Fcitx's code quality and coding style received some criticism, which was a prelude to Yuking stopping Fcitx development. Yuking restarted development from 2008.10.8.

For more related details, you can check the links at the end of this page.
