# Contribute to this Wiki

## Contributing to this project

Welcome! This documentation project provides documentation support for Fcitx 5 users, developers, and other contributors. We warmly welcome your contributions to this project!

You can contribute by submitting Issues or Pull Requests.

## Submitting Issues

If you find errors, omissions, or content that needs updating in the documentation, feel free to submit an Issue. Please ensure the Issue title clearly describes the problem and provide detailed information in the body.

## Submitting Pull Requests

### Fork the Repository

Click the Fork button on the GitHub page to copy the repository to your GitHub account.

### Clone the Repository

```bash
git clone https://github.com/wjjsn/fcitx-doc.git
cd fcitx-doc
```

### Create a Branch

```bash
git checkout -b your-branch-name
```

### Install Dependencies

```bash
npm install
```

### Preview Locally

```bash
npm run dev
```

### Commit Changes

```bash
git add .
git commit -m "Describe your changes"
git push origin your-branch-name
```

### Create Pull Request

Open your repository on GitHub, click the Compare & pull request button. Fill in the PR description, explaining what changes you made and why.

## Documentation Format

This project is built with Rspress, and documentation is written in Markdown format. You can use standard Markdown syntax.
