# Donate

For privacy reasons, we will only list the date and amount for all donations here (donation history).

All Fcitx developers are developing Fcitx in their spare time. So donating some money will be a great encouragement for them.

Currently, we don't have a centralized account to raise funds, so the only way to donate to Fcitx is to donate to individual developers.

All developers please feel free to add your name here. To protect privacy, all email addresses should not be displayed directly. If you want to find email accounts not listed here, you can also use your Google account to browse (http://code.google.com/p/fcitx Fcitx Google Code homepage), then click developers and contributors on each page, you can easily find their email accounts.

Since there is some other development happening on github, you can also simply clone the code and check email account records through git at (http://github.com/fcitx/ Fcitx Github organization)
